# AI-SDK-ANTHROPIC

[![AI-SDK Ecosystem](https://img.shields.io/badge/AI--SDK-ECOSYSTEM-part%20of-blue)](https://github.com/mk-knight23/AI-SDK-ECOSYSTEM)

> **Framework**: Anthropic Claude

Production-ready Anthropic Claude SDK implementation with modern web stack.

## Tech Stack

- **Frontend**: Remix
- **Backend**: FastAPI (Python)
- **AI SDK**: Anthropic Claude

## Quick Start

```bash
# Frontend
cd app
npm install
npm run dev

# Backend
cd backend
pip install -r requirements.txt
python main.py
```

## Features

- Claude API with message streaming
- RESTful API endpoints
- Docker containerization
- CI/CD pipeline
- Test coverage configured
